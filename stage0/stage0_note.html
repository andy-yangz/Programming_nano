My HTML notes

Part1: Technological empathy to understand what happens when we open a web page.

Several important concepts about internet

  • Web: Web to connect different computers. Most famous one is WWW (World Wide Web)
  • HTML: Hyper Text Markup Language. Main part of WWW.
  • urls: What we can use it to access website.
  • HTTP: Hyper Text Transfer Protocal. How html file be tranfered and displayed
  • Web Application: Applications used on web.

    • How these works?

    At first, user should use the browser, a web application, on his computer try to access a web site, which by entering urls in brower. And through web, by HTTP, the server will send us HTML files. So we can look these html files on our own computer through the brower.

    What make up HTML files

    There are almost four parts compose HTML files. They are:

  • Text Content: Text strings what we see.
  • Mark Up: What a web page looks like.
  • Reference to other documents: Such like images, videos, and audios.
  • Other Pages: We all familiar with this. Through hyperlink we can jump to other pages.

    Part2: Learning some tags to create a simple html page

    Elements

    HTML files are made by various elements.
    And normally elements are made by three parts, opening tag, content, closing tag. However, some tags can also have attributes. And even there are some tags called void tag with out content and closing tag.
    There are some examples.

  • Normal: <b> Bold; <em> Emphasis;
  • With Attributes <a href=url> Anchor
  • Void Tag: <img src=url> This image tag. It's have no closing tag because has no content.
    • More details.

    Inline & Block

    In addition to above variety, there are also a feature called Block is important. Block means a part of region divid with other parts. And we can change features of it's look like or something else. However, Inline is elements in a part of region or block. One easy way to discriminate them is to see will it change line, except <br> element. Example:

  • Block:<p>, <span>, <div>
  • Inline:<em> <br>

    • HTML file detail

  • Doctype: There is a line to declare document type. Normally, HTML.
  • Head: This part will describe the layout of this html file. It can have CSS, Javascript.
  • Body: Main part of this html file. What we want show other people will be there.

    • Summary

  • Normally, we go to WWW access HTML files through HTTP protcol.
  • HTML is made up by Text Content, Mark Up, and Reference to various documents and other pages.
  • HTML has many elements which have several categories. Like void tag, tag with attributes, and incline and block.
  • HTML has two parts. Head to tell the design. And body to tell the content.
